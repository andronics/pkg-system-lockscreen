# Lockscreen Manager - Screen locking and idle control

**Package**: `@system/lockscreen`
**Version**: 0.1.0
**Repository**: `pkg-system-lockscreen`

## Overview

Lockscreen Manager - Screen locking and idle control

## Installation

### Prerequisites

No external dependencies required.

### Using pkg-cli

```bash
# Install from GitHub
pkg-cli install @system/lockscreen

# Or install from local source
cd ~/.pkgs
stow lockscreen
```

## Usage

### System Integration

This package provides system integration for lockscreen manager - screen locking and idle control.


## Configuration

Configuration files are typically stored in:
- `~/.config/lockscreen/` - User configuration
- `~/.local/share/lockscreen/` - Application data

## Uninstallation

```bash
# Using pkg-cli
pkg-cli uninstall @system/lockscreen

# Or manual unstow
cd ~/.pkgs
stow -D lockscreen
```
